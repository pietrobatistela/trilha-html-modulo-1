O que é Semântica?

(Voltar)

A semântica é o estudo do significado, de forma mais clara, pode-se dizer que é dar sentido ao conteúdo.

Como isso se aplica em HTML?

(Voltar)

No HTML, temos o que chamamos de elementos semânticos, que são elementos com um significado, ou seja, ele claramente descreve seu significado tanto para o navegador quanto para o desenvolvedor.

Principais elementos

(Voltar)

Há diversos elementos ou tags semânticas, as mais utilizadas são: header, nav, main, article, aside, figure, section e footer.
